# 🖼️ SimpleFormGUI

A small form desktop applicaiton built with **C#** and **Avalonia UI** to learn.

## ✨ Features
- Designed with Avalonia to run on Windows, macOS, and Linux.
- Modern UI separation using AXAML for a sleek look.
- Intuitive forms for inputting and processing user profiles.
- Logic-driven checks to ensure data integrity.

## 🚀 How to Run
1. **Clone the repository**:
   `git clone https://github.com/maillet-nills/SimpleFormGUI.git`
2. **Open in your IDE**: (Visual Studio, JetBrains Rider, or VS Code).
3. **Run**: Use `dotnet run` or press `F5` to launch the application!

## 🛠️ Technologies
- **C# / .NET** 🟦
- **Avalonia UI** (XAML) 🖼️
